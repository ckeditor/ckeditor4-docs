Integration with forms (ngModel)

The CKEditor 4 WYSIWYG editor component implements the ControlValueAccessor interface and works with ngModel. For example, it can be used to integrate with a simple form like the one below.

User profile form

Editor data preview

Note that this is only a proof of concept of using ngModel. It allows for editing the form, but the editor strips out unknown tags. It also autoparagraphs text outside of block elements. The editor updates when the textarea element is blurred.
