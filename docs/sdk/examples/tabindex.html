Page Navigation Using the "Tab" Key Documentation

If keyboard navigation on a page is your thing, you will be happy to learn that CKEditor can be easily reached by using the Tab key, just like any other field element on the web page. This is the default behavior that does not require any configuration on your part.

Use the following keyboard shortcuts to navigate through the sample form below:

  • Tab – Move to the next page element.
  • Shift+Tab – Move to the previous page element.

Sample checkbox

Additionally, you can influence the place that CKEditor will assume in the web page Tab order. Refer to the Page Navigation Using the "Tab" Key article to learn more about this feature.
